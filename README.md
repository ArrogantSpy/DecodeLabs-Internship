# Data Science Internship/Certification Tasks

This repository contains my completed Data Science internship/certification tasks.

The project uses an e-commerce dataset to demonstrate basic data science skills, including dataset understanding, data cleaning, exploratory data analysis, data visualization, and business insight generation.

## Dataset

The dataset is an e-commerce order dataset provided by the certification organization.

It includes information such as:

- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Payment Method
- Order Status
- Coupon Code
- Referral Source
- Total Price

## Tasks Completed

### Task 1: Data Collection and Dataset Understanding

Reviewed the dataset structure, columns, data types, missing values, duplicate rows, and overall purpose.

### Task 2: Data Cleaning and Preprocessing

Cleaned the dataset by checking missing values, duplicates, formatting issues, and invalid values. Missing values in the CouponCode column were replaced with "No Coupon".

### Task 3: Exploratory Data Analysis

Analyzed product orders, revenue, order status, payment methods, coupon usage, and referral sources.

### Task 4: Data Visualization

Created charts to visually represent product orders, revenue by product, order status distribution, payment methods, referral sources, and coupon usage.

### Task 5: Business Insight and Recommendation Project

Generated business recommendations based on product performance, revenue, marketing channels, coupon usage, payment methods, and order status.
